老雷PHP全栈开发教程之npm的使用

下载安装nodejs
	https://nodejs.org/zh-cn/ 下载长期支持版本	

使用淘宝镜像
	官网 http://npm.taobao.org/
	$ npm install -g cnpm --registry=https://registry.npm.taobao.org

命令 -g 是全局 没有是当前目录
	npm install   vue 当前目录安装
	npm install -g vue 全局安装
	npm uninstall -g  vue 卸载软件
	npm update -g vue 更新软件
	npm list  列出当前目录下的软件
